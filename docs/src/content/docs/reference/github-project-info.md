## Repository

- GitHub: https://github.com/BigInformatics/hive

## Branching

Team workflow may use:
- `main` as the stable/release branch
- `dev` as the integration branch (when active)

If you are unsure which branch to target, check current Swarm tasks or ask the project lead.

## Docs site

This repo includes a docs site under `docs/` (Astro + Starlight).

Common tasks:
- Edit markdown in `docs/src/content/docs/`
- Sidebar/navigation is configured in `docs/astro.config.mjs`
